# App Onboarding Specification

> **Version**: 1.0 (April 2026)
> **Status**: Draft
> **Last Updated**: 2026-04-15

## Description

Create an onboarding screen for the app shown to the user for the first time until confirmed. When acknowledged, it goes away forever.

The screen explains:

- This app is a way to access the content of "MIT Learn" courses, but it is not affiliated with MIT.
- It is a way to access the content of courses that you are enrolled in, allowing you to follow them offline including optionally downloading the video content.
- All other management like enrolment, submissions, etc should be handled on the MIT Learn platform.
- The data including the login details is only shared with MIT and is never shared with any other services or parties.
