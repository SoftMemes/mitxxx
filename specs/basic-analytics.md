# Basic Analytics Specification

> **Version**: 1.0 (April 2026)
> **Status**: Draft
> **Last Updated**: 2026-04-15

## Description

Using the Firebase/Firestore support for Google Analytics, add tracking events for core actions in the app. Cover all basic interactions with custom events, including:

**App-level events:**
- Opening the app
- Successfully logging in
- Triggering a sync (app-level or individual course)
- Triggering a download (per level)

**Course-level events:**
- Viewing a course

**In-course interaction events:**
- Play start/stop
- Scrubbing
- Opening a section
- Hitting play from a section
